# Simple-File-System
A simple file system implemented in C

## Requirement
The implementation of SFS consists two parts: three filesystem-related functions
- open_t(),
- read_t(), and
- write_t()),
and seven commands (
    - mkfs_t,
    - ls_t,
    - cd_t,
    - mkdir_t,
    - external_cp,
    - cp_t,
    - cat_t).
