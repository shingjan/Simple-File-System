

Compile
==========================
gcc mkfs.c -o mkfs_t
gcc tshell.c -o tshell


Run
==========================
./mkfs_t HD
./tshell




Files
==========================

mkfs.c: File System
tshell.c: Shell System
f1.h: Struct Definition
f2.h: Method Details
f3.h: open_t, read_t, write_t method
f4.h: mkfs_t, ls_t, cd_t, mkdir_t, external_cp, cp_t, cat_t method
